# Projectgeheugen

Gedeelde, gecureerde kennis voor dit project. De `itworxs-*` agents **lezen** deze
bestanden voor context en werken ze **bewust** bij wanneer ze iets duurzaams leren.
Houd het kort en accuraat — dit is geen logboek. Verwijder verouderde info.

- `architecture.md` — stack, structuur, belangrijke beslissingen, bekende valkuilen, db-schema.
- `domain.md` — glossarium: afkortingen, klant- en projecttermen.
- `data-privacy.md` — PII-inventaris en privacy-aandachtspunten (beheerd door `itworxs-gdpr`).

Deze bestanden blijven behouden bij `itworxs update` — ze worden nooit overschreven.
